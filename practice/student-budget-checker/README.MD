# Student Budget Checker

A simple Excel budgeting tool for tracking monthly income and expenses.

## Features
- Monthly income calculator
- Expense tracking
- Remaining balance calculation
- Expense bar chart visualization

## Skills Used
- Excel formulas
- SUM function
- Cell references
- Spreadsheet formatting
- Basic data visualization

## Files
- `student-budget-checker.xlsx`

## Preview
This project compares part-time income with monthly spending to help students manage their budget effectively.
